  • HTML&CSS

    HTML과 CSS는 웹페이지의 골격을 만들고 꾸미는 데에 사용되는 언어들이다.

  • JavaScript

    JavaScript는 브라우저 또는 Node환경에서 동작하는 프로그래밍 언어로, 웹페이지의 동작을 담당한다.

  • React

    React는 현재 가장 점유율이 높은 JavaScript 라이브러리로, 컴포넌트를 이용하여 웹개발의 효율을 향상시킨다.

  • NodeJS

    NodeJS는 Chrome의 V8엔진을 이용한 JavaScript 런타임 환경으로, JavaScript를 브라우저 외의 환경에서도 동작하게 만들어준다.

  • TypeScript

    TypeScirpt는 Microsoft에서 만든 JavaScript의 superset으로, 예상되는 타입을 암시해주고 개발 과정에서 오류를 미리 잡아주는 등의 편리함을 가져다 준다.
